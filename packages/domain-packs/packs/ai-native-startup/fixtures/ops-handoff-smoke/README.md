# Ops Handoff Smoke Fixture

This fixture represents the scale-stage operating artifacts Runstead should
govern after MVP and launch readiness are working.

Expected evidence:

- founder bottleneck map names founder-only decisions and knowledge
- workflow registry records recurring operations and automation coverage
- SOP artifacts identify owners, inputs, outputs, and approval points
- support triage evidence links customer reports to remediation tasks
- GTM artifacts cite customer or metric evidence before publication
